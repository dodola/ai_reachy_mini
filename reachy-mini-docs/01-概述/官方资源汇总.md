## 中国用户特别说明

### 访问 Hugging Face
使用镜像：https://hf-mirror.com/

设置环境变量：
```bash
export HF_ENDPOINT=https://hf-mirror.com/
```

### 对话应用在中国的使用
需要配置 VPN 或使用代理访问 OpenAI API。

详见 [故障排除指南](../07-故障排除/常见问题与解决方案.md)
