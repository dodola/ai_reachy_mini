# Reachy Mini 产品介绍

## 概述

**Reachy Mini** 是首款开源桌面机器人，旨在探索人机交互和创意自定义应用。该机器人由 **Pollen Robotics** 和 **Hugging Face** 共同开发，由 **Seeed Studio** 优化和制造。

## 核心特性

### 富有表现力的运动
- **6自由度头部：** 实现自然的平移、倾斜和滚动运动
- **全机身旋转：** 360 度旋转实现动态定位
- **灵动天线：** 2个富有表现力的天线，增加个性化反馈

### 多模态感知
- **4麦克风阵列：** 先进的音频捕获，用于语音识别和声源定位
- **广角摄像头：** 120° 广角，12MP，自动对焦
- **5W 扬声器：** 清晰的音频输出

### 开放且可扩展
- **完全开源：** 硬件设计、软件 SDK 和仿真环境全部公开
- **Python SDK：** 直观的编程接口
- **Hugging Face 集成：** 直接访问最先进的 AI 模型

## 硬件版本对比

| 特性 | Reachy Mini Lite ($299) | Reachy Mini Full ($449) |
|------|------------------------|------------------------|
| **计算单元** | 外部设备 (Mac/Linux) | 板载 Raspberry Pi 4 |
| **连接性** | 仅限有线 USB | WiFi + 以太网 |
| **电源** | 有线供电 (7V-5A) | 电池 + 有线操作 |
| **麦克风** | 4个 | 4个 |
| **扬声器** | 5W | 5W |
| **摄像头** | 广角 120° | 广角 120° |
| **加速计** | 无 | 有 (IMU) |
| **头部运动** | 6自由度 (DOF) | 6自由度 (DOF) |
| **身体旋转** | 是 (360°) | 是 (360°) |
| **天线** | 2个 | 2个 |
| **高度** | 28 cm (11") | 28 cm (11") |
| **重量** | 1.5 kg (3.3 lbs) | 1.5 kg (3.3 lbs) |

## 物理尺寸

- **高度：** 28 cm / 11 英寸 (休眠模式下约为 23 cm / 9 英寸)
- **宽度：** 16 cm / 6.3 英寸
- **重量：** 1.5 kg / 3.3 磅

## 适用场景

### Reachy Mini Lite
- 使用 Mac/Linux 系统的开发者
- 希望最大程度控制计算资源的用户
- 拥有现有计算机实验室的教育机构
- 需要与现有基础设施集成的项目

### Reachy Mini Full
- 独立机器人应用
- 无线研究与开发
- 需要加速计数据的项目
- 移动演示和部署

## 应用生态系统

Reachy Mini 拥有不断发展的应用生态系统，由 Hugging Face Spaces 提供支持：

- **Conversation App** - 自然语言交互
- **Hand Tracker** - 实时手部追踪
- **Radio** - 语音控制的电台流媒体
- **更多应用** - 在 Hugging Face Spaces 上浏览

## 隐私与安全

- **无数据采集：** 默认不存储、传输或处理个人数据
- **本地处理：** 所有 AI 处理均可在本地完成
- **用户控制：** 摄像头和麦克风功能完全由用户控制
- **开源：** 软件和硬件完全透明
