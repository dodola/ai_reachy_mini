# Reachy Mini 快速入门指南

## 前置要求

确保已按照 [SDK 安装指南](./SDK安装指南.md) 安装了 Reachy Mini SDK。

> **重要:** 确保已创建并激活 Python 虚拟环境。每次打开新终端时都要激活它！

## 1. 启动 Daemon

Daemon 是处理电机和传感器通信的后台服务。它必须运行，您的代码才能工作。

### Reachy Mini (Wireless)
- 开机时 daemon 自动运行
- 确保电脑和机器人在同一网络

### Reachy Mini Lite (USB)
有两种方式:

**方式 1: 使用桌面应用**
- 下载并启动 Reachy Mini Control 应用

**方式 2: 命令行**
```bash
reachy-mini-daemon
```

### 仿真模式 (无需硬件)
```bash
# Linux & Windows
reachy-mini-daemon --sim

# macOS
mjpython -m reachy_mini.daemon.app.main --sim
```

> **macOS 用户注意:** `uv` 可能与 MuJoCo 有兼容性问题。如果遇到问题，建议使用 `pip` 代替 `uv`。

### 验证 Daemon 运行

打开浏览器访问: http://localhost:8000/docs

如果看到 Reachy SDK API 文档，说明 daemon 已成功运行！

## 2. 连接机器人

### 自动连接 (推荐)

SDK 会自动检测连接类型:
- **Lite 版本:** 通过 USB 连接到本地 daemon
- **Wireless 版本:** 通过网络连接到机器人

```python
from reachy_mini import ReachyMini

with ReachyMini() as mini:
    print("已连接到 Reachy Mini!")
```

### 手动指定连接模式

```python
# 强制本地连接 (Lite)
with ReachyMini(connection_mode="localhost_only") as mini:
    pass

# 强制网络连接 (Wireless)
with ReachyMini(connection_mode="network") as mini:
    pass
```

## 3. 第一个脚本

创建文件 `hello.py`:

```python
from reachy_mini import ReachyMini

# 连接到运行中的 daemon
with ReachyMini() as mini:
    print("已连接到 Reachy Mini! 🤖")
    
    # 摆动天线
    print("摆动天线...")
    mini.goto_target(antennas=[0.5, -0.5], duration=0.5)
    mini.goto_target(antennas=[-0.5, 0.5], duration=0.5)
    mini.goto_target(antennas=[0, 0], duration=0.5)

    print("完成!")
```

运行脚本:
```bash
python hello.py
```

🎉 如果一切顺利，机器人会摆动它的天线！

## 4. 更多示例

### 移动头部

```python
from reachy_mini import ReachyMini
from reachy_mini.utils import create_head_pose

with ReachyMini() as mini:
    # 向上看并倾斜头部
    mini.goto_target(
        head=create_head_pose(z=10, roll=15, degrees=True, mm=True),
        duration=1.0
    )
    
    # 向下看
    mini.goto_target(
        head=create_head_pose(z=-10, mm=True),
        duration=1.0
    )
    
    # 转头
    mini.goto_target(
        head=create_head_pose(yaw=30, degrees=True),
        duration=1.0
    )
```

### 获取摄像头画面

```python
from reachy_mini import ReachyMini

with ReachyMini(media_backend="default") as mini:
    # 获取一帧图像
    frame = mini.media.get_frame()
    print(f"图像形状: {frame.shape}")  # (height, width, 3)
```

### 录制音频

```python
from reachy_mini import ReachyMini
import time

with ReachyMini(media_backend="default") as mini:
    # 开始录制
    mini.media.start_recording()
    
    # 录制 3 秒
    time.sleep(3)
    
    # 获取音频样本
    samples = mini.media.get_audio_sample()
    print(f"音频形状: {samples.shape}")  # (samples, 2)
    
    # 停止录制
    mini.media.stop_recording()
```

### 播放音频

```python
from reachy_mini import ReachyMini
import numpy as np

with ReachyMini(media_backend="default") as mini:
    # 开始播放
    mini.media.start_playing()
    
    # 生成正弦波
    sample_rate = 16000
    duration = 1.0
    t = np.linspace(0, duration, int(sample_rate * duration))
    audio = np.sin(2 * np.pi * 440 * t)  # 440 Hz
    audio = np.column_stack([audio, audio])  # 立体声
    
    # 播放
    mini.media.push_audio_sample(audio.astype(np.float32))
    time.sleep(duration)
    
    # 停止播放
    mini.media.stop_playing()
```

## 7. 远程连接到 Wireless 版本

### SSH 连接到机器人

```bash
ssh pollen@reachy-mini
# 用户名: pollen
# 密码: root
```

### 在机器人上运行代码

```bash
# 激活虚拟环境
source /venvs/apps_venv/bin/activate

# 运行脚本
python your_script.py
```

## 8. 使用 REST API

Daemon 提供 HTTP/WebSocket API:

```bash
# 获取机器人状态
curl http://localhost:8000/api/state/full

# 查看 API 文档
open http://localhost:8000/docs
```

### WebSocket 实时更新

```javascript
let ws = new WebSocket('ws://127.0.0.1:8000/api/state/ws/full');

ws.onmessage = (event) => {
    const data = JSON.parse(event.data);
    console.log(data);
};
```
