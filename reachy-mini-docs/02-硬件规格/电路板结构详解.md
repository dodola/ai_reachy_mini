# Reachy Mini 硬件电路板结构详解

## 概述

Reachy Mini 由多个电路板组成，负责电源管理、电机控制、传感器接口和通信功能。

## 电路板架构

### 整体结构

```
┌─────────────────────────────────────────────────────────────┐
│                      Reachy Mini                            │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐         │
│  │ 摄像头模块  │  │ 麦克风阵列  │  │  扬声器     │         │
│  │ (CSI)       │  │ (I2C/USB)   │  │  (PWM)      │         │
│  └──────┬──────┘  └──────┬──────┘  └──────┬──────┘         │
│         │                │                │                 │
│         └────────────────┼────────────────┘                 │
│                          │                                  │
│                    ┌─────▼─────┐                            │
│                    │ 控制板    │                            │
│                    │ (主控)    │                            │
│                    └─────┬─────┘                            │

└─────────────────────────────────────────────────────────────┘
```

## 主要电路板

### 1. 电源板 (Power Board)

**功能:**
- 电源输入和分配
- 电池管理 (Wireless 版本)
- 电压转换
- 保护电路

**规格:**
```
输入电压: 6.8 - 7.6V
输出电压: 5V, 3.3V (为其他板供电)
电池类型: LiFePO4 (Wireless)
容量: 2000mAh, 6.4V, 12.8Wh
```

**保护功能:**
- ✅ 过充保护
- ✅ 过放保护
- ✅ 过流保护
- ✅ 短路保护
- ✅ 温度传感器

**接口:**
```
┌─────────────────────────────────────┐
│         Power Board                 │
├─────────────────────────────────────┤
│ - 电源输入 (DC 6.8-7.6V)          │
│ - 电池接口 (Wireless)              │
│ - USB-C 充电接口 (Wireless)        │
│ - 电源输出 (5V, 3.3V)             │
│ - 电机电源输出                     │
│ - LED 指示灯 (绿/橙/红)           │
│ - 温度传感器接口                   │
└─────────────────────────────────────┘
```

**连接器:**
- **电源输入:** DC Jack 或 USB-C
- **电池接口:** JST 或焊接
- **电源输出:** 杜邦线或 FPC

---

### 2. 控制板 (Controller Board)

#### Lite 版本控制板

**功能:**
- USB 通信接口
- 电机 TTL 转换
- 传感器数据采集

**规格:**
```
输入电压: 6.8-7.6V (来自电源板)
通信接口: USB-C (数据)
电机接口: TTL (Dynamixel)
摄像头接口: CSI
麦克风接口: I2C/FPC
```

**接口:**
```
┌─────────────────────────────────────┐
│      Lite Controller Board          │
├─────────────────────────────────────┤
│ - 电源输入 (6.8-7.6V)             │
│ - USB-C 输出 (数据，不充电)        │
│ - Dynamixel TTL 接口               │
│ - Camera CSI 接口                  │
│ - 麦克风阵列接口                   │
│ - 扬声器接口                       │
└─────────────────────────────────────┘
```

#### Wireless 版本控制板 (CM4 Board)

**功能:**
- Raspberry Pi CM4 接口
- WiFi/蓝牙通信
- 独立计算能力

**规格:**
```
输入电压: 6.8-7.6V (来自电源板)
CM4 型号: CM4104016
WiFi: 2.4-5GHz 双频
蓝牙: 5.0 BLE
RAM: 4GB
存储: 16GB eMMC
```

**接口:**
```
┌─────────────────────────────────────┐
│       CM4 Controller Board          │
├─────────────────────────────────────┤
│ - 电源输入 (6.8-7.6V)             │
│ - CM4 接口 (200-pin)               │
│ - USB-C 输出 (数据，不充电)        │
│ - Dynamixel TTL 接口               │
│ - Camera CSI 接口                  │
│ - 麦克风阵列接口                   │
│ - WiFi 天线接口 (U.FL)             │
│ - 扬声器接口                       │
└─────────────────────────────────────┘
```

---

### 3. 麦克风阵列板 (Mic Array Board)

**功能:**
- 音频采集
- 声源定位 (DoA)
- 波束成形
- 回声消除

**规格:**
```
型号: reSpeaker XMOS XVF3800
麦克风: 4x PDM MEMS 数字麦克风
采样率: 16 kHz
灵敏度: -26 dB FS
信噪比: 64 dBA
接口: I2C/USB
```

**布局:**
```
┌─────────────────────────────────────┐
│       Mic Array Board               │
│                                     │
│         MIC1    MIC2                │
│            ┌───┐                    │
│            │   │                    │
│         MIC3    MIC4                │
│                                     │
│    ┌─────────────────────────┐     │
│    │ XMOS XVF3800 处理器     │     │
│    └─────────────────────────┘     │
│                                     │
│    [FPC 接口] [I2C/USB 接口]       │
└─────────────────────────────────────┘
```

**连接:**
- **FPC 线:** 12-pin, 0.5mm 间距, Type A
- **数据接口:** I2C 或 USB

---

### 4. 摄像头模块 (Camera Module)

**功能:**
- 图像采集
- 视频流
- 计算机视觉

**规格:**
```
型号: Raspberry Pi Camera v3 Wide Angle
传感器: Sony IMX708
分辨率: 12MP
视角: 120° (广角)
对焦: 自动对焦
帧率: 30 FPS
接口: CSI (Camera Serial Interface)
```

**连接:**
- **CSI 线:** 15-pin, 1mm 间距
- **供电:** 3.3V (来自控制板)

---

### 5. 电机控制电路

**电机类型:**
```
┌─────────────────────────────────────┐
│          Motor Layout               │
├─────────────────────────────────────┤
│ 底座: 1x XC330-M288-PG (ID: 10)   │
│ Stewart 平台: 6x XL330-M288-T      │
│   - ID: 11, 12, 13, 14, 15, 16    │
│ 天线: 2x XL330-M077-T              │
│   - ID: 17 (右), 18 (左)          │
└─────────────────────────────────────┘
```

**通信协议:**
```
协议: TTL (Dynamixel)
波特率: 1,000,000 bps
电压: 3.3V
连接器: JST 3-pin
```

**电机连接:**
```
┌─────────────────────────────────────┐
│       Motor Connection              │
├─────────────────────────────────────┤
│                                     │
│   [控制板] ──TTL──► [电机]        │
│                                     │
│   3 线连接:                         │
│   - VCC (电源)                      │
│   - GND (地)                        │
│   - DATA (数据)                     │
│                                     │
│   线缆长度:                         │
│   - 底座: 300mm                     │
│   - Stewart: 200mm                  │
│   - 天线: 100mm                     │
│   - 短线: 40mm                      │
└─────────────────────────────────────┘
```

---

### 6. IMU 传感器模块 (仅 Wireless)

**功能:**
- 惯性测量
- 姿态检测
- 运动检测

**规格:**
```
类型: 6 轴 IMU
加速度计: 3 轴 (±16g)
陀螺仪: 3 轴 (±2000°/s)
接口: I2C
采样率: 100-1000 Hz
```

**数据输出:**
```python
{
    "accelerometer": (x, y, z),  # m/s²
    "gyroscope": (x, y, z),      # rad/s
    "quaternion": (w, x, y, z),
    "temperature": float          # °C
}
```

## 接口类型详解

### 1. TTL 接口 (Dynamixel)

**用途:** 电机通信

**引脚定义:**
```
┌─────────────────┐
│  JST 3-pin      │
├─────────────────┤
│ Pin 1: VCC      │ (电源, 5V)
│ Pin 2: GND      │ (地)
│ Pin 3: DATA     │ (数据, 3.3V TTL)
└─────────────────┘
```

**通信参数:**
- 波特率: 1,000,000 bps
- 协议: Dynamixel Protocol 2.0
- 电压: 3.3V TTL

---

### 2. CSI 接口 (摄像头)

**用途:** 摄像头数据传输

**引脚定义:**
```
┌─────────────────┐
│  15-pin FPC     │
├─────────────────┤
│ Pin 1-2: GND    │
│ Pin 3-6: DATA   │ (差分数据线)
│ Pin 7: CLK      │ (时钟)
│ Pin 8: GND      │
│ Pin 9-10: 3.3V  │ (电源)
│ Pin 11-15: 其他 │
└─────────────────┘
```

---

### 3. I2C 接口 (传感器)

**用途:** 传感器通信

**引脚定义:**
```
┌─────────────────┐
│  4-pin          │
├─────────────────┤
│ Pin 1: VCC      │ (3.3V 或 5V)
│ Pin 2: GND      │ (地)
│ Pin 3: SDA      │ (数据)
│ Pin 4: SCL      │ (时钟)
└─────────────────┘
```

---

### 4. FPC 接口 (柔性电路)

**用途:** 麦克风、传感器连接

**规格:**
- 12-pin
- 0.5mm 间距
- Type A (连接器在同一侧)
- 15mm 长度

## 故障排除

### 常见电路板问题

#### 1. 电源板问题

**症状:**
- 无法开机
- LED 不亮
- 电机不响应

**检查:**
```
1. 检查电源输入电压 (6.8-7.6V)
2. 检查电源线连接
3. 检查保险丝
4. 检查电源开关
```

---

#### 2. 控制板问题

**症状:**
- 无法连接电脑
- 无法连接电机
- 摄像头不工作

**检查:**
```
1. 检查 USB 线
2. 检查 TTL 线连接
3. 检查 CSI 线连接
4. 检查电源指示灯
```

---

#### 3. 麦克风板问题

**症状:**
- 无音频输入
- 音频质量差
- 声源定位失败

**检查:**
```
1. 检查 FPC 线方向 (蓝色面朝上)
2. 检查 FPC 线是否损坏
3. 检查 I2C/USB 连接
```

---

#### 4. 电机问题

**症状:**
- 电机不响应
- 电机闪烁红灯
- 电机抖动

**检查:**
```
1. 检查 TTL 线连接
2. 检查电机电源
3. 检查电机 ID
4. 扫描电机
```

## 维护建议

### 日常维护

1. **检查连接**
   - 定期检查所有线缆连接
   - 确保连接器牢固

2. **清洁**
   - 保持电路板清洁
   - 避免灰尘积累

3. **散热**
   - 确保通风良好
   - 避免长时间高负载

### 故障预防

1. **电源管理**
   - 使用原装电源适配器
   - 避免电压波动

2. **线缆管理**
   - 避免线缆过度弯曲
   - 定期检查线缆磨损

3. **静电防护**
   - 触摸电路板前先接地
   - 使用防静电手环
