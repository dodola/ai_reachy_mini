# Reachy Mini Lite Controller Board 接口详解

## 概述

Lite Controller Board 是 Reachy Mini Lite 版本的核心控制板，负责电机控制、传感器数据采集和与外部电脑的通信。

## 后面板接口

从官方图片可以看到，后面板有以下接口:

```
┌─────────────────────────────────────────────────────────────┐
│                    Reachy Mini 后面板                        │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   ┌─────────┐  ┌─────────┐  ┌─────────┐  ┌─────────┐     │
│   │Dynamixel│  │  Arm    │  │ USB-C   │  │ 7V 5A   │     │
│   │  (X)    │  │         │  │ Data    │  │ Power   │     │
│   └─────────┘  └─────────┘  └─────────┘  └─────────┘     │
│                                                             │
│   ┌─────────┐  ┌─────────────────────────────────────┐   │
│   │ Power   │  │         Battery Status              │   │
│   │ Switch  │  │  ● ● ● (绿/橙/红)                  │   │
│   └─────────┘  └─────────────────────────────────────┘   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 接口说明

| 接口 | 类型 | 功能 | 规格 |
|------|------|------|------|
| **Dynamixel (X)** | JST 3-pin | 电机 TTL 连接 | 3.3V, 1M bps |
| **Arm** | FPC/杜邦 | 手臂接口 | 保留/扩展 |
| **USB-C Data** | USB-C | 数据连接 | USB 2.0, 不充电 |
| **7V 5A Power** | DC Jack | 电源输入 | 7V, 5A |
| **Power Switch** | 开关 | 电源开关 | - |
| **Battery Status** | LED | 电池状态 | 绿/橙/红 |

## 是否有外部接口可以在内部接入开发板？

### 短回答

**⚠️ 有限制，但有几种可能的方案**

---

### 方案分析

#### 方案 1: USB-C 接口 (最简单)

**可行性:** ✅ 推荐

```
┌─────────────┐     USB      ┌─────────────┐
│ Lite        │◄────────────►│ Raspberry   │
│ Controller  │              │ Pi 4        │
└─────────────┘              └─────────────┘
```

**实现方式:**
- 使用 USB 线连接 Lite Controller Board 的 USB-C 接口到 Raspberry Pi
- Raspberry Pi 运行 daemon
- 通过 WiFi 远程控制

**优点:**
- ✅ 无需改造硬件
- ✅ 保持保修
- ✅ 简单可靠

**缺点:**
- ❌ 需要外部线缆
- ❌ 占用 USB-C 端口

---

#### 方案 2: 内部 USB 跳线 (需要改造)

**可行性:** ⚠️ 需要电子技能

```
┌─────────────┐     USB 跳线     ┌─────────────┐
│ Lite        │◄────────────────►│ Raspberry   │
│ Controller  │   (内部连接)     │ Pi 4        │
└─────────────┘                  └─────────────┘
```

**实现方式:**
1. 打开机器人外壳
2. 找到 Lite Controller Board 上的 USB 测试点或接口
3. 焊接 USB 跳线
4. 连接到内部 Raspberry Pi

**可能的连接点:**

```
┌─────────────────────────────────────┐
│      Lite Controller Board          │
│                                     │
│   [USB-C 接口]                      │
│        │                            │
│        ▼                            │
│   [USB 转换芯片]                    │
│        │                            │
│        ▼                            │
│   [测试点/焊盘] ◄── 可能的接入点   │
│                                     │
│   [TTL 驱动芯片]                    │
│        │                            │
│        ▼                            │
│   [电机接口]                        │
│                                     │
└─────────────────────────────────────┘
```

**风险:**
- 🔴 可能损坏主板
- 🔴 失去保修
- 🔴 需要精确焊接

---

#### 方案 3: 串口/UART 接口 (高级)

**可行性:** ⚠️ 需要深入研究

如果 Lite Controller Board 有未使用的 UART 接口:

```
┌─────────────────────────────────────┐
│      Lite Controller Board          │
│                                     │
│   [主控芯片]                        │
│        │                            │
│        ├──► USB (已使用)            │
│        │                            │
│        ├──► UART (可能未使用) ◄─── 可能的接入点
│        │                            │
│        └──► SPI/I2C (可能未使用)    │
│                                     │
└─────────────────────────────────────┘
```

**实现方式:**
1. 找到 UART 测试点
2. 连接 Raspberry Pi 的 GPIO
3. 配置串口通信

**优点:**
- ✅ 不占用 USB 端口
- ✅ 可以同时充电和数据传输

**缺点:**
- ❌ 需要了解主板设计
- ❌ 可能需要修改固件

---

#### 方案 4: 外部 Hub (推荐)

**可行性:** ✅ 推荐

```
┌─────────────┐     USB      ┌─────────────┐     USB      ┌─────────────┐
│ Lite        │◄────────────►│ USB Hub     │◄────────────►│ Raspberry   │
│ Controller  │              │             │              │ Pi 4        │
└─────────────┘              └─────────────┘              └─────────────┘
```

**实现方式:**
1. 在内部放置一个小型 USB Hub
2. Hub 连接 Lite Controller Board
3. Raspberry Pi 连接到 Hub
4. 可以同时连接其他 USB 设备

**优点:**
- ✅ 扩展 USB 端口
- ✅ 可以连接多个设备
- ✅ 无需改造主板

**缺点:**
- ❌ 需要额外空间
- ❌ 需要供电

---

## 实际改造建议

### 最佳方案: 外接 Raspberry Pi + 内部布线

```
┌─────────────────────────────────────────────────────────────┐
│                    Reachy Mini 内部改造                      │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   ┌─────────────────────────────────────────────────────┐ │
│   │                 底座空间                            │ │
│   │                                                     │ │
│   │   ┌─────────────┐      ┌─────────────┐            │ │
│   │   │ Raspberry   │ USB  │ USB Hub     │            │ │
│   │   │ Pi 4        │◄────►│ (小型)      │            │ │
│   │   └─────────────┘      └──────┬──────┘            │ │
│   │                                │                    │ │
│   │                                │ USB                │ │
│   │                                ▼                    │ │
│   │                         ┌─────────────┐            │ │
│   │                         │ Lite        │            │ │
│   │                         │ Controller  │            │ │
│   │                         └─────────────┘            │ │
│   │                                                     │ │
│   └─────────────────────────────────────────────────────┘ │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 实现步骤

#### 步骤 1: 评估空间
```bash
# 打开外壳，测量底座空间
# 通常有 50-80mm x 50-80mm 的空间
```

#### 步骤 2: 选择组件
```
Raspberry Pi 4 (85mm x 56mm)
小型 USB Hub (30mm x 20mm)
短 USB 线 (10-15cm)
```

#### 步骤 3: 布线
```
1. 将 Raspberry Pi 放在底座
2. USB Hub 连接到 Lite Controller Board
3. Raspberry Pi 连接到 USB Hub
4. 电源线连接到 Raspberry Pi
```

#### 步骤 4: 固定
```
1. 使用双面胶或螺丝固定
2. 确保不干扰机械运动
3. 保持散热空间
```

## 电源考虑

### 电源需求

| 设备 | 电压 | 电流 |
|------|------|------|
| **Lite Controller** | 7V | 1.5A |
| **Raspberry Pi 4** | 5V | 3A |
| **USB Hub** | 5V | 0.5A |

### 供电方案

#### 方案 A: 独立供电 (简单)
```
墙壁插座 ──► 7V 5A 适配器 ──► Lite Controller
墙壁插座 ──► 5V 3A 适配器 ──► Raspberry Pi 4
```

#### 方案 B: 内部供电 (紧凑)
```
墙壁插座 ──► 7V 5A 适配器 ──► Lite Controller
                         │
                         └──► DC-DC 转换器 ──► 5V ──► Raspberry Pi 4
```

**DC-DC 转换器规格:**
- 输入: 6.8-7.6V
- 输出: 5V, 3A
- 尺寸: 小型模块

## 社区经验

### 参考项目

1. **Lite + RPi4 方案**
   - 外接 Raspberry Pi 4
   - 通过 USB 连接
   - 简单可靠

2. **内部集成方案**
   - 在底座放置 RPi4
   - 使用短 USB 线
   - 需要改造布线

### 注意事项

1. **散热**
   - Raspberry Pi 4 会产生热量
   - 需要通风空间
   - 可能需要散热片

2. **干扰**
   - USB 线可能干扰电机
   - 使用屏蔽线
   - 避免靠近电机

3. **固定**
   - 确保不会松动
   - 不要干扰机械运动
   - 使用防震材料

## 总结

### 接口可用性

| 方案 | 可行性 | 难度 | 推荐度 |
|------|--------|------|--------|
| **外部 USB 连接** | ✅ | ⭐ | ⭐⭐⭐⭐⭐ |
| **内部 USB 跳线** | ⚠️ | ⭐⭐⭐ | ⭐⭐ |
| **UART 接口** | ⚠️ | ⭐⭐⭐⭐ | ⭐⭐ |
| **外部 Hub** | ✅ | ⭐⭐ | ⭐⭐⭐⭐ |

### 推荐方案

**初学者:** 外部 USB 连接 (最简单)
**进阶用户:** 内部放置 RPi4 + USB Hub (紧凑)
**高级用户:** 内部 USB 跳线 (最集成)

### 最佳实践

1. **先尝试外部连接** - 验证功能
2. **再考虑内部集成** - 优化布线
3. **保持可逆性** - 方便恢复

**外部 USB 连接是最简单、最可靠的方案！** 🔌
