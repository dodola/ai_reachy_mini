# Reachy Mini 电机系统详解

## 电机概述

Reachy Mini 使用 Dynamixel 系列伺服电机，提供精确的位置控制和反馈。

## 电机列表

| 位置 | 型号 | ID | 说明 |
|------|------|-----|------|
| 底座 | XC330-M288-PG | 10 | 身体旋转 |
| Stewart 平台 1 | XL330-M288-T | 11 | 头部运动 |
| Stewart 平台 2 | XL330-M288-T | 12 | 头部运动 |
| Stewart 平台 3 | XL330-M288-T | 13 | 头部运动 |
| Stewart 平台 4 | XL330-M288-T | 14 | 头部运动 |
| Stewart 平台 5 | XL330-M288-T | 15 | 头部运动 |
| Stewart 平台 6 | XL330-M288-T | 16 | 头部运动 |
| 左天线 | XL330-M077-T | 17 | 天线运动 |
| 右天线 | XL330-M077-T | 18 | 天线运动 |

## 电机规格

### XC330-M288-PG (底座)
- **类型:** 塑料齿轮版本
- **扭矩:** 0.86 Nm (连续)
- **速度:** 77 RPM
- **通信:** TTL
- **分辨率:** 4096 步/圈

### XL330-M288-T (Stewart 平台)
- **扭矩:** 0.52 Nm (连续)
- **速度:** 77 RPM
- **通信:** TTL
- **分辨率:** 4096 步/圈

### XL330-M077-T (天线)
- **扭矩:** 0.14 Nm (连续)
- **速度:** 228 RPM
- **通信:** TTL
- **分辨率:** 4096 步/圈
- **特点:** 速度更快，适合天线的快速响应

## 通信协议

### TTL 通信
- **波特率:** 1,000,000 bps
- **电压:** 3.3V
- **连接器:** JST 3-pin

### 电机扫描
使用以下命令扫描电机：

```bash
# Lite 版本
python -m reachy_mini.tools.scan_motors

# Wireless 版本
ssh pollen@reachy-mini.local
source /venvs/mini_daemon/bin/activate
python -m reachy_mini.tools.scan_motors --wireless
```

正常输出示例：
```
Trying baudrate: 1000000
Found motors at baudrate 1000000: [10, 11, 12, 13, 14, 15, 16, 17, 18]
```

## 电机控制

### 位置控制
```python
from reachy_mini import ReachyMini
from reachy_mini.utils import create_head_pose

with ReachyMini() as mini:
    # 移动头部
    mini.goto_target(
        head=create_head_pose(z=10, roll=15, degrees=True, mm=True),
        duration=1.0
    )
    
    # 移动天线
    mini.goto_target(antennas=[0.5, -0.5], duration=0.5)
    
    # 移动身体
    mini.goto_target(body_yaw=30, degrees=True, duration=1.0)
```

### 电机模式
```python
with ReachyMini() as mini:
    # 启用电机 (僵硬模式)
    mini.enable_motors()
    
    # 禁用电机 (松弛模式)
    mini.disable_motors()
    
    # 启用重力补偿
    mini.enable_gravity_compensation()
```

## PID 调优

如果电机抖动，可以调整 PID 参数：

```yaml
# hardware_config.yaml
motors:
  10:  # 底座
    P: 180
    I: 0
    D: 10
  17:  # 左天线
    P: 180
    I: 0
    D: 10
  18:  # 右天线
    P: 180
    I: 0
    D: 10
```

### 调优建议
1. 首先尝试降低 P 值到 180
2. 如果仍有抖动，增加 D 值到 10
3. 不同机器人可能需要不同的参数

## 天线抖动问题

### 原因
天线在垂直位置 (0°) 时容易抖动，因为齿轮间隙导致不稳定平衡。

### 解决方案
1. **偏移天线位置:** 将天线偏移几度 (通常 10° 足够)
2. **调整 PID:** 降低 P 值或增加 D 值

### 默认行为
Reachy Mini 现在默认将天线偏移几度以避免抖动。

## 电机错误

### 常见错误
- **过载错误:** 电机负载过大
- **输入电压错误:** 电压不在范围内 (正常: 6.8-7.6V)
- **过热错误:** 电机温度过高

### 错误处理
```python
with ReachyMini() as mini:
    # 检查电机状态
    status = mini.client.get_status()
    print(status)
    
    # 重启电机
    mini.disable_motors()
    mini.enable_motors()
```

## 电机维护

### 日常维护
- 保持电机清洁
- 避免过载
- 定期检查连接

### 故障排除
1. 检查电缆连接
2. 验证电源供应
3. 扫描电机确认通信
4. 检查电机 ID 和波特率
