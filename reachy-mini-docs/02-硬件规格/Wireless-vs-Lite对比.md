# Reachy Mini Wireless vs Lite 硬件对比

## 📊 核心区别一览

| 特性 | Reachy Mini Lite | Reachy Mini Wireless |
|------|------------------|---------------------|
| **价格** | $299 | $449 |
| **计算单元** | 外部计算机 (Mac/Linux) | 板载 Raspberry Pi 4 |
| **连接方式** | USB 有线 | WiFi + 以太网 |
| **电源** | 有线供电 (7V-5A) | 电池 + 有线 |
| **独立性** | 需要连接电脑 | 完全独立运行 |
| **IMU/加速计** | ❌ 无 | ✅ 有 |

---

## 🔧 详细硬件对比

### 1. 计算单元

#### Lite 版本
```
┌─────────────┐     USB-C     ┌─────────────┐
│ Reachy Mini │◄────────────►│  您的电脑   │
│   (本体)    │   (仅数据)   │ (Mac/Linux) │
└─────────────┘              └─────────────┘
```
- **计算:** 依赖外部电脑
- **优势:** 可使用电脑的全部算力
- **限制:** 必须有线连接

#### Wireless 版本
```
┌─────────────────────────────┐
│ Reachy Mini                 │
│ ┌─────────────────────────┐ │
│ │ Raspberry Pi CM4        │ │
│ │ - CPU: ARM Cortex-A72   │ │
│ │ - RAM: 4GB              │ │
│ │ - 存储: 16GB Flash      │ │
│ │ - WiFi: 2.4/5GHz 双频   │ │
│ └─────────────────────────┘ │
└─────────────────────────────┘
```
- **计算:** 板载 Raspberry Pi Compute Module 4
- **型号:** CM4104016
- **WiFi:** 支持 (2.4-5GHz 双频)
- **天线:** 2.4-5GHz 双频贴片天线, 2.79 dBi

---

### 2. 电源系统

#### Lite 版本
```
┌─────────────┐     USB-C     ┌─────────────┐
│ 墙插电源    │──────────────►│ Reachy Mini │
│ 7V-5A       │   (仅数据)   │             │
└─────────────┘              └─────────────┘
```
- **供电:** 7V-5A 电源适配器
- **连接:** USB-C (仅数据，不充电)
- **特点:** 必须插电使用

#### Wireless 版本
```
┌─────────────┐     USB-C     ┌─────────────┐
│ Reachy Mini │◄────────────►│ 充电器/电脑 │
│             │   (可充电)   │             │
│ ┌─────────┐ │              └─────────────┘
│ │ 电池    │ │
│ │ 2000mAh │ │
│ │ LiFePO4 │ │
│ └─────────┘ │
└─────────────┘
```
- **电池类型:** LiFePO4 (磷酸铁锂)
- **容量:** 2000mAh
- **电压:** 6.4V
- **能量:** 12.8Wh
- **续航:** 2-4 小时
- **充电:** USB-C
- **保护功能:**
  - ✅ 过充保护
  - ✅ 过放保护
  - ✅ 过流保护
  - ✅ 短路保护
  - ✅ 温度传感器

---

### 3. 连接方式

#### Lite 版本
```
┌─────────────┐     USB      ┌─────────────┐
│ Reachy Mini │◄────────────►│  您的电脑   │
└─────────────┘   (有线)     └─────────────┘
```
- **连接:** USB-C 有线
- **协议:** USB 2.0
- **距离:** 受 USB 线长度限制 (~2-3米)

#### Wireless 版本
```
┌─────────────┐     WiFi      ┌─────────────┐
│ Reachy Mini │◄─────────────►│  您的电脑   │
│             │    (无线)     │  或手机     │
└─────────────┘               └─────────────┘
         │
         │ 以太网 (可选)
         ▼
    ┌─────────┐
    │ 路由器  │
    └─────────┘
```
- **连接:** WiFi 无线
- **WiFi 标准:** 2.4-5GHz 双频
- **备用:** 以太网 (通过 USB-C 适配器)
- **访问:** `reachy-mini.local` (mDNS)

### 4. 传感器系统

#### 两者共有
| 传感器 | 规格 |
|--------|------|
| **摄像头** | Raspberry Pi Camera v3 Wide Angle |
| **传感器** | Sony IMX708 |
| **分辨率** | 12MP |
| **视角** | 120° (广角) |
| **对焦** | 自动对焦 |
| **帧率** | 30 FPS |
| **麦克风** | 4x PDM MEMS 数字麦克风 |
| **采样率** | 16 kHz |
| **扬声器** | 5W @ 4 Ohms |

#### Wireless 独有
| 传感器 | 规格 |
|--------|------|
| **IMU** | 6 轴惯性测量单元 |
| **加速度计** | 3 轴 (m/s²) |
| **陀螺仪** | 3 轴 (rad/s) |
| **四元数** | (w, x, y, z) |
| **温度** | 传感器温度 (°C) |

```python
# Wireless 版本 IMU 数据示例
with ReachyMini() as mini:
    imu = mini.imu
    accel = imu["accelerometer"]  # (ax, ay, az) m/s²
    gyro = imu["gyroscope"]       # (gx, gy, gz) rad/s
    quat = imu["quaternion"]      # (w, x, y, z)
    temp = imu["temperature"]     # °C
```

## 🎯 使用场景对比

- 远程控制
- 多房间使用
- 便携展示

- 需要 IMU 数据
- 姿态检测
- 运动分析

## 🛠️ 开发体验对比

### Lite 版本
```bash
# 1. 连接 USB
# 2. 启动 daemon
reachy-mini-daemon

# 3. 运行代码
python your_script.py
```
**优势:**
- 更强的计算能力
- 直接访问硬件
- 更低的延迟

### Wireless 版本
```bash
# 1. 开机自动运行 daemon
# 2. 连接 WiFi
# 3. SSH 或远程运行
ssh pollen@reachy-mini.local

# 或直接使用 SDK
python your_script.py  # 自动连接
```
**优势:**
- 无需 USB 线
- 独立运行
- 可远程控制

## 🎮 代码兼容性

两个版本的代码完全兼容，SDK 会自动检测版本：

```python
from reachy_mini import ReachyMini

# 自动检测连接类型
with ReachyMini() as mini:
    # 通用代码
    mini.goto_target(antennas=[0.5, -0.5], duration=0.5)
    
    # 检查是否为 Wireless 版本
    if hasattr(mini, 'imu'):
        # Wireless 独有功能
        imu_data = mini.imu
        print(f"加速度: {imu_data['accelerometer']}")
```

## 🔍 如何选择？

### 选择 Lite 如果：
- 💻 您有 Mac/Linux 电脑
- 💰 预算有限 ($299)
- 🔌 不介意有线连接
- 🖥️ 需要完整计算能力
- 🏫 教育机构使用

### 选择 Wireless 如果：
- 📱 需要独立运行
- 📶 需要无线连接
- 🎯 需要 IMU 传感器
- 🚀 想要移动演示
- 💵 预算充足 ($449)
